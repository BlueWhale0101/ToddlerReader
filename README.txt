# Pony Reading Adventure PWA

This package contains a Progressive Web App version of the pony reading game.

## Files
- `index.html`
- `styles.css`
- `app.js`
- `manifest.webmanifest`
- `service-worker.js`
- icons

## Deploy
Upload the whole folder to any static host with HTTPS, such as GitHub Pages or Netlify.

## Install on iPhone
Open the deployed site in Safari, tap Share, then choose **Add to Home Screen**.

## Notes
Progress is saved locally in the browser on each device.
